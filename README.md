# Enhanced Calculator

A command-line calculator application developed in Python using advanced object-oriented programming principles, design patterns, pandas, and automated testing.

## Features
- REPL interface
- Arithmetic operations
- Undo/Redo
- History persistence
- pandas integration
- Factory, Strategy, Observer, Memento, and Facade patterns
- 100% test coverage

## Run the Application

```bash
python -m app.calculator_repl
